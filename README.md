## 📘 Description

This repository contains my full solutions to the programming assignments for the course **"Advanced Java Programming" (course number 20554)** at the University, semester **Spring 2024**.

The course covers a wide range of advanced Java topics, including:

- Object-Oriented Programming and class hierarchies  
- JavaFX GUI development  
- Collections and generics (ArrayList, LinkedList, Map)  
- Exception handling  
- Threads and concurrency (synchronization, simulation)
- Custom data structures and algorithms  

All assignments were implemented, tested, and submitted through the university's platform, and the final grade received was **100**.
